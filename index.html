Política de Privacidade

Última atualização: [14/12/2024]

Introdução

Sua privacidade é importante para nós. Este documento explica como coletamos, usamos e protegemos suas informações ao utilizar nosso aplicativo.

Informações Coletadas

Nosso aplicativo utiliza a API do Gmail para realizar ações específicas, como buscar e-mails baseados em critérios e excluí-los. Durante o uso do aplicativo, os seguintes dados podem ser acessados:

  • Metadados dos e-mails (assunto, remetente, data e hora).
  • IDs das mensagens para operações específicas, como exclusão.

Não armazenamos nem compartilhamos os conteúdos dos e-mails.

Uso de Informações

As informações acessadas por meio da API do Gmail são usadas exclusivamente para os seguintes fins:

  1. Processar solicitações específicas feitas por você (ex.: apagar e-mails).
  2. Garantir o funcionamento adequado do aplicativo.

Compartilhamento de Informações

Nós não compartilhamos, vendemos ou alugamos suas informações com terceiros. Todo o processamento de dados ocorre localmente no seu dispositivo.

Segurança

Levamos a segurança a sério. As informações acessadas por meio da API do Gmail são protegidas pelas credenciais OAuth, e nenhuma informação é armazenada fora do ambiente do aplicativo.

Conformidade com a API do Gmail

Nosso aplicativo segue as Políticas de Dados do Usuário da API do Google, incluindo:

  • O uso adequado de dados apenas para fins autorizados.
  • A exclusão imediata de dados após o uso.

Links Externos

Nosso aplicativo pode conter links para serviços externos. Não somos responsáveis pelas práticas de privacidade de sites ou serviços de terceiros.

Alterações nesta Política

Podemos atualizar esta política de privacidade ocasionalmente. Recomendamos que você revise este documento periodicamente para estar ciente de quaisquer mudanças.
